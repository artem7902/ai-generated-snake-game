# 🐍 AI Generated Snake Game

🤖 This project is an AI-generated version of the classic Snake game.

📱 It was created with React Native and Expo, and it also has a web version:

🌐 https://ai-generated-snake-game.vercel.app

✨ The project was created as proof that the AI era is already here.

## ⚙️ Commands

- 🚀 `yarn start` - starts the Expo development server.
- 🤖 `yarn android` - starts the app on Android.
- 🍎 `yarn ios` - starts the app on iOS.
- 🌐 `yarn web` - starts the web version locally.
- 📦 `yarn build:web` - builds the web version into the `dist` folder.
